# Banker
Banker MCBE Add-On adds to a new type of trader that will buy various rare items or accumulated in exchange for emeralds, thus activating the game economy.

The banker will help to reward activities of the game that are not always rewarded as copper mining, natural carbon obtaining, book manufacturing and beekeeping.

This addon is optimized to run effectively on Realms and dedicated servers with minimal hardware resource consumption and its available in english and español. 🇬🇧 🇲🇽
